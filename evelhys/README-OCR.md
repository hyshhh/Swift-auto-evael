# 弦号识别 Pipeline

> 基于 Qwen3-VL-8B 的船体弦号自动识别系统

## 一句话总结

> **输入图片 → 调用大模型识别弦号位置 → 输出标注图片和坐标**

---

## 快速开始

```bash
# 1. 安装依赖
pip install requests pillow tqdm

# 2. 运行识别
python OCR.py --image_dir ./images

# 3. 查看结果
# 结果保存在 ./images/results/
```

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 自动识别 | 调用 Qwen3-VL-8B 识别弦号位置 |
| 坐标输出 | 输出归一化坐标 (x, y) |
| 可视化标注 | 在图片上绘制矩形框和中心点 |
| 批量处理 | 支持整个目录的图片批量识别 |
| 结果汇总 | 自动生成 JSON 汇总文件 |

## 使用方法

### 基础用法

```bash
python OCR.py --image_dir G:/秋招/weitiao/ms-swift/evelhys/images
```

### 完整参数

```bash
python OCR.py \
    --image_dir ./images \
    --output_dir ./results \
    --box_size 150 \
    --line_width 4 \
    --model Qwen/Qwen3-VL-8B-Instruct
```

---

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--image_dir` | （必填） | 图片目录路径 |
| `--output_dir` | None | 输出目录（默认 image_dir/results） |
| `--api_key` | 预置 | API Key |
| `--api_url` | SiliconFlow | API 地址 |
| `--model` | Qwen3-VL-8B | 模型名称 |
| `--box_size` | 100 | 标注框大小（像素） |
| `--line_width` | 3 | 标注线宽度 |

## 提示词说明

```
你是一个专业的弦号识别专家。

任务：识别图片中船体侧面的弦号位置，输出其中心点的归一化坐标。

要求：
1. 弦号位于船体侧面（船舷外侧），为数字/字母组合标识
2. 优先关注船体侧面区域，忽略背景文字、水印、标牌等干扰
3. 即使模糊、遮挡、低分辨率，也必须尽量估计其中心位置
4. 每张图像中必定存在且仅存在一个弦号

输出格式（严格遵守）：
{"x": float, "y": float}
```

## 常见问题

### Q1: API 调用失败

检查：
- API Key 是否正确
- 网络连接是否正常
- 图片大小是否超过限制（建议 < 10MB）

### Q2: 识别位置不准确

可能原因：
- 图片中没有弦号
- 弦号被严重遮挡
- 图片分辨率太低

解决方案：
- 确保图片包含船体侧面
- 提高图片分辨率
- 调整提示词

### Q3: 输出坐标是 (0.5, 0.5)

说明模型未能识别到弦号，返回了默认中心点。检查：
- 图片是否包含弦号
- API 响应是否正确
